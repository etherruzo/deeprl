The assignment¶

is to implement a deep reinforcement learning algorithm in python3 and pytorch where the agent successfully plays Banana Game, a game where the agent has to collect yellow bananas while avoiding blue bananas. I have adapted DQN with experienced replay algorithm and vary some of the hyperparameters to obtain the best performance.

What is DQN with experienced replay?¶

DQN algorithm uses two separate networks with identical architectures. For this assignment I have used two fully connected layers and the choice to include batch normalization. The target Q-networks weights are updated less often than the primary Q-network weights, specifically 1 update every 4 iterations, in order to avoid correlation between the parameters we are looking for and the parameters of the target. O

Another improvement with respect to vanilla DQN in order to avoid correlation between the sequence of experience tuples and helping the agent to learn, so that it does not get stack in tuples the agent is no longer learning from. Experienced replay keeps track of a replay buffer (memory) , and then sample from this buffer at random. Another advantage of experienced replay is that it allows us to learn more from individual tuples multiple times

from matplotlib import pyplot as plt
import pandas as pd
import numpy as  np


df=pd.read_pickle('finaldf_r.p')

dic_dfs={}

eps_ini=[1,0.5]
eps_fin=[0.1,0.01]
max_ts=[1000,10000,50000]

fig,ax=plt.subplots(1,figsize=(12,6))
for maxt in max_ts:
    dic_dfs[maxt]={}
    for epsfin in eps_fin:
        dic_dfs[maxt][epsfin]={}
        for epsini in eps_ini:
            dic_dfs[maxt][epsfin][epsini]={}
            sub=df[(df.maxt==maxt)&(df.eps1==epsini)&(df.eps2==epsfin)].r.values[0]
            x=list(range(len(sub)))
            ax.plot(x,sub,label='eps1_'+str(epsini)+'_eps2_'+str(epsfin)+'_maxt_'+str(maxt))
ax.legend()
ax.set_xlim([0,505])
plt.xlabel("Number of episodes");
plt.ylabel("Score");

Next steps would be to try to change the experienced replay update. In the bibliography I've seen different values (larger than the ones used here) to see if the agent improves. I would also try to implement prioritized experienced replay algorithm. This means trying to change the sampling distribution by defining priority experience items based on the largest difference between our predictions and targets (since these corresponds to the samples that we know the least , and therefore have a great path for improvement).
